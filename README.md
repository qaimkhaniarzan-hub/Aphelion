This is Aphelion, a Lunar Lander game which i made myself, completely contained in a single HTML file.
Current features:
-> Procedurally infinitely generating Land
-> A mode for docking (Quite buggy)
-> Missions for credits
-> Upgrades to buy with the credits

Features I am currently working on:
-> Scroll Zoom
-> More lander models
-> Multiplayer support
-> Levels to docking, each with a higher level of difficulty
-> Procedurally generating sattelites for infinite levels of docking
-> An OST as background music

Features I hope to implement in the future
-> Building your own rocket
-> Launching from Earth
-> Unique scenarios meant to try to survive
-> Avatars (for astronauts)
-> A skin system

Thats pretty much it! Leave comments to give me some ideas
